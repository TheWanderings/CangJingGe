#### 数据库主键ID获取
* URL: `/ids/ids_generate`
* Method:   GET
* 功能描述：根据表名获取一定数量的数据库主键ID。
* 请求参数： 
     
| 参数名称   | 参数类型  |是否必须 | 描述                                    
| ----------|---------- |------| ---------------------------|
| type   | string   |  是 | 数据库表名 |
| count   | int   |  是 | 需要的id数量|


* 返回参数：

* Example:

```json

     [100, 101, 102, .....]

```
